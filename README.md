# Raccoon Tools Packages

Monorepo for reusable Python packages published to PyPI. Each package lives in its own directory with independent versioning, dependencies, and CI workflows.

## Packages

- [Raccoon Tools Utils](raccoontools/README.md): General-purpose utilities, retry decorators, HTTP helpers, file operations, serialization, and generators.
- [Raccoon Tools DB](raccoontools-db/README.md): Managed PostgreSQL connection pool with opt-in FastAPI and OpenTelemetry integrations.
